# System Design Interview Prep

Daily system design interview questions and answers for Staff+ engineers with 20+ years of experience.

Each question includes:
- Problem statement with requirements and constraints
- Architecture diagrams
- Deep dive into components
- Scalability and trade-off analysis
- Failure modes and resilience
- Follow-up questions

## Format

Files are organized by date:
```
questions/
├── 2025-02-27-global-fraud-detection.md
├── 2025-02-28-distributed-key-value-store.md
└── ...
```

## Usage

Each file contains a complete question + answer suitable for:
- Interview preparation
- Study groups
- Architecture discussions
- Technical deep-dives
